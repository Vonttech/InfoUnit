InfoUnit
Unidade da Informação - projeto de rede social

Sobre o projeto:
O projeto de rede social consiste em um meio de comunicação entre estudantes para facilitar o compartilhamento de conhecimento entre eles, de forma que se um estudante tem dificuldade em assuntos de uma disciplina ele poderia encontrar alguém na rede social para auxiliá-lo nos estudos. A rede social também reforça a metodologia de mentoria na instituição expandindo esta metodologia além dos muros do campus.

Requisitos:
Alguns requisitos para o InfoUnit são essenciais, pois estes são os que originaram a ideia de uma rede social. Os requisitos são:



“Match”UBER - Uma pessoa deve ser capaz de publicar uma demanda e esta demanda deve ser atendida por outra pessoa que se responsabilizará pelo cumprimento, com direito à alguma recompensa que contará para sua carreira acadêmica ou penalidade no caso de desfeita.
Postagens - As pessoas podem publicar postagens de algum determinado assunto de natureza acadêmica. 
Histórico - O perfil da pessoa deve conter um histórico de suas atividades na plataforma.
Filtro - Deve existir a possibilidade de filtragem de assuntos que interessam ao usuário.
Chat - Deve existir um chat em tempo real entre os usuários da plataforma.
Favoritos - Uma filtragem generalizada, entre pessoas, posts, etc.
Administração - Deve haver uma hierarquia de administradores e moderadores e áreas para que ambos acessem.
